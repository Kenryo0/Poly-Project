# But du jeu

Réussir à sortir du donjon en interagissant avec l'environnement et en actionnant des mécanismes.

## Contrôles

Déplacement : **ZQSD** pour se mouvoir et **La souris** pour déplacer la caméra.
Interaction : 
- **E** pour interagir avec les objets et ramasser les rochers
- **F** pour lâcher les rochers

HAVE FUN !
